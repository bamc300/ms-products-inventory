# Diseño de Arquitectura

Este directorio contiene información relacionada con el diseño visual y arquitectónico del sistema.

## Origen del Diseño

Los diagramas de arquitectura y flujo presentados en este proyecto fueron diseñados originalmente utilizando la herramienta **Draw.io**.

### Justificación

Se eligió Draw.io para la fase de diseño inicial debido a:
1.  **Flexibilidad Visual**: Facilita la creación rápida de diagramas de componentes y secuencias sin la rigidez del código.
2.  **Colaboración**: Permite compartir y revisar visualmente la arquitectura antes de la implementación.
3.  **Referencia**: Estos diseños sirven como la fuente de verdad visual que posteriormente se tradujo a diagramas Mermaid en el `README.md` principal para mantener la documentación cerca del código.

## Archivos Fuente

Si se dispone de los archivos `.drawio` originales, se recomienda almacenarlos en esta carpeta para facilitar futuras modificaciones visuales.
